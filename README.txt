----- Instrucciones --------

1. Ejecutar en la consola (Inicio > Ejecutar "cmd"), dentro de la carpeta de la aplicación (cd c:\xxxx\xxxxx\) ruby init.rb
2. Una vez que esté funcionando, se tiene que asegurarse que este logueado al instagram, abrir el navegador y escribir "http://localhost:4567/login" una vez que se ingresen los datos, va a llevar directamente a la consola de mando
3. En otra ventana se tiene que ingresar a "http://localhost:4567/" para ver la aplicación funcionando.
4. Para reniciar o limpiar la base de datos, tiene que ingresar a "http://localhost:4567/resetdb"
5. Para cambiar el hashtag se tiene que ingresar en el navegador a "http://localhost:4567/hashtag/<el hashtag nuevo>", el hashtag no tiene que contener espacios, ni caracteres especiales.
